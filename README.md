# HangMan

Game of hang man

# Technologies

## Backend

- Node.js
- Postgress
- Express
- MQTT

## Frontend

- React
- SCSS
- WebSocket

# Setup

- Install and configure locally MQTT server.
- Run:

```
cd backend
npm install
npm start
```

- Run:

```
cd frontend
npm install
npm start
```

# Usage

- Open http://localhost:3000 in your browser.
- Create a new game.
